🚀 Cara Daftar & Setup OpenRouter

Dapatkan $1 GRATIS untuk generate soal dengan AI!

🎉 TIDAK PERLU KARTU KREDIT! Cukup Google/GitHub account

⚠️ PENTING: OpenRouter memberikan $1 free credits untuk akun baru. Cukup untuk ~50-100x generate soal (250-500 soal total)!

1 Buka OpenRouter

Klik link ini atau copy ke browser:

🔗 Buka OpenRouter.ai

URL: https://openrouter.ai

2 Sign In

Klik tombol "Sign In" di pojok kanan atas

Pilih salah satu metode login:

  • ✅ Continue with Google (Recommended - Tercepat!)
  • ✅ Continue with GitHub
  • ✅ Continue with Discord
  • ✅ Email (perlu verifikasi email)

💡 TIP: Gunakan Google untuk paling cepat!

3 Verifikasi & Welcome Page

Setelah login, Anda akan masuk ke dashboard OpenRouter.

Anda akan melihat:

  • 💰 Free credits: $1.00
  • 📊 Usage dashboard
  • 🔑 Keys menu

4 Buat API Key

Cara 1 - Via Dashboard:

  1. Klik icon profile (pojok kanan atas)
  2. Pilih "Keys"

Cara 2 - Direct Link:

🔑 Langsung ke Keys Page

URL: https://openrouter.ai/keys

5 Create New Key

Di halaman Keys:

  1. Klik tombol "Create Key"
  2. Beri nama: DashboardGuru
  3. Limit (opsional): Kosongkan saja (unlimited)
  4. Klik "Create"

6 Copy API Key

⚠️ PENTING: API key hanya ditampilkan SATU KALI!

  1. API key akan muncul dengan format: sk-or-v1-xxxxxxxxx...
  2. Klik tombol "Copy" atau select & copy manual
  3. Simpan di tempat yang aman (atau langsung paste ke .env)
⚠️ NOTE: Jika Anda kehilangan API key, Anda harus membuat yang baru. API key tidak bisa dilihat lagi setelah dialog ditutup.

7 Paste ke File .env

Buka file .env di project DashboardGuru Anda

Cari baris ini:

OPENAI_API_KEY=

Paste API key Anda:

OPENAI_API_KEY=sk-or-v1-xxxxxxxxxxxxxxxxxxxxxxxxx

💾 SAVE file .env

8 Test Koneksi

Buka terminal/command prompt di folder project, jalankan:

php test-openrouter.php

Jika berhasil, Anda akan melihat:

✅ SUCCESS! API is working!
Response: Halo dari DashboardGuru!
Tokens used: X tokens

9 Test Generate Soal

Sekarang buka aplikasi DashboardGuru:

  1. Login sebagai Guru
  2. Masuk ke Bank Soal → Soal Tambahan
  3. Klik tombol hijau "Generate Soal dengan AI"
  4. Isi form:
    • Ilustrasi Materi: (deskripsi materi)
    • Jenis Soal: Pilihan Ganda / Essay
    • Tingkat Kesulitan: Mudah
    • Jumlah Soal: 3 (mulai kecil dulu)
  5. Klik "Generate Soal dengan AI"
  6. Tunggu 5-10 detik...
  7. ✅ SUKSES! Soal akan muncul untuk preview & edit

🎉 SELESAI!

Selamat! Fitur AI Question Generator sudah siap digunakan!

Anda punya $1 free credits untuk:

  • ~50-100x generate (5 soal per generate)
  • Total: 250-500 soal gratis!

💡 Tips & Tricks

  • Start Small: Generate 3-5 soal dulu untuk test
  • Deskripsi Detail: Semakin detail ilustrasi, semakin bagus hasilnya
  • Edit Hasil: Review & edit soal sebelum disimpan
  • Monitor Credit: Cek usage di OpenRouter Activity
  • Model Gratis: Jika credit habis, bisa ubah ke model google/gemini-pro (gratis selamanya!)

🔧 Troubleshooting

❌ Error: "AI API key belum dikonfigurasi"

  • Pastikan sudah paste API key ke .env
  • Jalankan: php artisan config:clear
  • Refresh browser / restart server

❌ Error: "Rate limit"

  • Tunggu 1-2 menit
  • Kurangi jumlah soal (3-5 saja)
  • Cek credit: OpenRouter Credits

❌ Error: "Insufficient credits"

  • Free $1 sudah habis
  • Add credit minimal $5 di OpenRouter
  • Atau ubah model ke google/gemini-pro (gratis)
